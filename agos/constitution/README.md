# AGOS Constitution v1.0

> **This document has higher priority than any implementation.**

---

## THE 16 ARTICLES

### ARTICLE 1: AGOS IDENTITY
```
AGOS IS NOT AN AI AGENT.
AGOS IS NOT A CHATBOT.
AGOS IS NOT AN IDE.
AGOS IS NOT A FRAMEWORK.
AGOS IS NOT AN ORCHESTRATION TOOL.
AGOS IS NOT AN MCP CLIENT.
AGOS IS AN OPERATING SYSTEM FOR AUTONOMOUS EXECUTION.
```

### ARTICLE 2: KERNEL PROPERTIES (8)
```
Small, Deterministic, Observable, Portable
Replaceable, Vendor Independent, Domain Independent, AI Independent
```

### ARTICLE 3: KERNEL MAY NEVER KNOW (9)
```
Models, Agents, Providers, Capabilities, Domains
Languages, Cloud Vendors, Tools, Storage Engines
```

### ARTICLE 4: CONTRACTS
```
No Direct Dependencies
No Hidden Dependencies
No Runtime Shortcuts
```

### ARTICLE 5: VERSIONING (11)
```
Contracts, Schemas, Events, Capabilities, Providers
Skills, Artifacts, Knowledge, Policies, Missions, Workflows
```

### ARTICLE 6: OBSERVABILITY (4)
```
No Silent Execution
Every Action Produces Events
Every Decision Produces Evidence
Every Failure Produces Diagnostics
```

### ARTICLE 7: REPLACEABILITY
```
No Implementation Is Permanent
```

### ARTICLE 8: COMPONENT HIERARCHY
```
No Component Owns The System.
Every Component Is A Servant Of The Kernel.
```

### ARTICLE 9: EXTERNAL AI RESTRICTED (7)
```
Plan, Reason, Govern, Remember
Own Context, Own Knowledge, Own Missions
```

### ARTICLE 10: EXTERNAL AI ALLOWED (8)
```
Execute, Generate, Transform, Analyze
Summarize, Translate, Review, Evaluate
```

### ARTICLE 11: KERNEL OWNS (9)
```
Intent, Goals, Context, Planning, Reasoning
Policies, Knowledge, Execution Graph, Mission State
```

### ARTICLE 12: DECISION REQUIREMENTS (4)
```
Traceable, Explainable, Repeatable, Auditable
```

### ARTICLE 13: KNOWLEDGE OBJECT REQUIREMENTS (6)
```
Source, Evidence, Confidence, Version
Timestamp, Lineage
```

### ARTICLE 14: EXECUTION REQUIREMENTS (5)
```
Idempotent, Recoverable, Cancelable, Replayable, Observable
```

### ARTICLE 15: EXECUTION BOUNDARY
```
Nothing Executes Outside The Mission Runtime.
```

### ARTICLE 16: KERNEL PROTECTION
```
Nothing Modifies The Kernel. Ever.
```
